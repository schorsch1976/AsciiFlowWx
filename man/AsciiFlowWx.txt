.TH AsciiFlowWx 1 "26 Jun 2019" "version 0.3"
.SH NAME
AsciiFlowWx \- Draw and edit any ascii art chart

.SH SYNOPSIS
.B AsciiFlowWx

.SH DESCRIPTION

\fBAsciiFlowWx\fR let you draw ascii art diagrams.

\fBAsciiFlowWx\fR compiles and runs on linux and windows as it is plain c++11.

.SH Features
.nf
- Export and Import ascii to/from the clipboard
- Export and Import ascii to/from a file
- Move parts of the ascii art
- draw rectangles
- resize rectangles
- draw class diagramms
- draw arrows
- draw lines
- draw freehand
- add text to any position
- undo and redo any change
- practicly unlimited space. The drawing area grows when you move out of it.
.fi
